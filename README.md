# WheatherWhere :sunny:

## [프로젝트 설명]
### :date: 개발기간
#### 2023.03.06 ~ 2023.04.13

### :pushpin: 기획의도
#### 대부분의 사람들이 외출하기 전에 참고하는 날씨와 일정 또 그에 맞게 뭘 입어야할지 결정할 때, 이 모든 것을 하나의 프로그램에서 확인할 수 있도록 Weather_Where를 기획하였다.

### ⚒️ 개발환경
#### Frontend
- HTML5
- Javascript
- jQuery 3.6.3
- CSS3
- Thymeleaf
#### Backend
- Java 11.0.17
- Spring Boot(Gradle) 2.7.9
#### DB
- Oracle Database 21c
- OpenAPI(기상청 단기예보 조회서비스 - https://www.data.go.kr/data/15084084/openapi.do)
#### ORM
- Mybatis
- JPA
#### Tools
- IntelliJ 2022.3.2
- DBeaver 22.3.0
- POSTMAN
- GitHub
- Figma

- 주소 CSV파일 파싱
